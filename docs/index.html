Common Questions

  • Can’t sign in? Make sure you’re on the latest app version and have a reliable network. If you recently changed devices, try logging out and back in.
  • Recent shows empty? Add a new transaction on Home to verify connectivity. If it still doesn’t appear, contact support with your device model and app version.
  • CSV/XLSX import fails? Ensure the file is under 25 MB and includes headers for date/description/amount. If you can, send a redacted sample to support so we can reproduce.
  • Duplicate transactions? The importer uses hash‑based matching to avoid duplicates. If you see unexpected duplicates, tell us the date/amount/description so we can investigate.
